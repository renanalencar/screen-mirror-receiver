# Screen Mirror Receiver

A simple web-based screen mirror receiver that serves a front-end UI for viewing mirrored content.

## Project Structure

- `index.html` - Main HTML page for the receiver UI.
- `css/styles.css` - Styles for the receiver page.
- `js/script.js` - JavaScript logic for the receiver.
- `.env` - Environment variables (ignored by Git).
- `.gitignore` - Files and folders excluded from version control.

## Usage

1. Open `index.html` in a browser.
2. Configure the sender or mirror device to stream to the receiver page.
3. Use the page as the screen mirror receiver interface.

## Notes

- This project is currently a static web app.
- Add a local web server if browser restrictions block local file access.
